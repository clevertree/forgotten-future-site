How to Contribute

There are many ways to contribute to the Forgotten Future Project.

Play the game

The first way is: play the game! Feedback (through anonymous statistics or the old fashioned way) helps the contributors improve the game over time. Subscribe to browser updates so you know when Forgotten Future is launched!

Sponsor the game (Make a donation)

The second way is: become an official sponsor by making a donation (in the amount of $0.01 or greater)! Forgotten Future is free to play for everyone forever. However, all sponsors get a little extra. For example, every sponsor who donates above a certain tier gets their username appear in the game credits, including console releases! Other tier rewards include products such as digital downloads or features like syncing saved data across devices. This is a good way of thanking our financial contributors.

Develop the game

The third way is: help develop the game! Forgotten Future is open-source, which means anyone can help work on the game visuals, code, and stage development. The source may also be forked and used as boiler plate for making your own games. That's how it should work, right? Contributors that donate above a certain amount may request developer access to the source repository. Here are some tips to get you started:

  • Forgotten Future Source Repository (github)
  • Beginners guide to Github
  • Edit and Preview Github Source Files
